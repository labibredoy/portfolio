Blood Donation Mobile App

Android application (branded LifeDrop) that connects blood donors and recipients in real time using geolocation, blood-type matching, and Firebase-backed secure authentication—reducing emergency response time when hospitals need compatible donors quickly.

> OBJECTIVES

  • Streamline blood donation by automating donor–recipient matching by blood group and proximity
  • Improve accessibility with an intuitive UI for users of all technical backgrounds
  • Notify suitable nearby donors instantly during medical emergencies
  • Promote regular donation through reminders and educational content on eligibility and benefits

> SYSTEM FLOW

Users register as donors or request blood with type and location. The app searches compatible donors, sends notifications, and supports accept/decline workflows with contact details surfaced after a match—documented in the project report as a full app diagram from signup through successful data update.

  • Welcome, login, and registration (name, phone, blood group, password)
  • Home hub: profile, donor search, request status, available donors
  • Educational info page on donation types and eligibility
  • Request pipeline with real-time status and notifications

> STACK & METHODOLOGY

Android Studio Java Firebase Auth & Database Geolocation

Development followed structured phases: conceptualization, UI design, iterative coding, QA testing, and Google Play deployment planning. Team project — Batch 16th, BAUET.
